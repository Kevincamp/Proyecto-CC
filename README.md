# Mi Dulce Arte (En desarrollo)
Este es un proyecto de la asignatura de Cloud Computing de la maestría de la universidad de Granada.

# Descripción del problema

Dulce Arte es una tienda de insumos de pastelería a pequeñas, medianas y grandes empresas, ahora tiene todo sus registros de productos, ventas, clientes y otros en papel. Dulce arte quiere actualizarse de manera que puedan llevar sus registros en linea, ademas quiza podrían mejorar sus ventas con el perfil de sus clientes.
